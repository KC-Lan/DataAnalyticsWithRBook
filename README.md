
#資料科學與R語言

本書網址: http://yijutseng.github.io/DataScienceRBook/index.html

[長庚大學資訊管理學系](http://im.cgu.edu.tw/bin/home.php) [大數據分析方法課程](https://github.com/yijutseng/BigDataCGUIM)教學使用書籍，內容包括使用R語言完成以下任務:

- 資料讀取 (檔案、透過API擷取或爬蟲)
- 資料清洗與處理 
- 探索式資料分析
- 資料視覺化
- 資料探勘
- R與Hadoop Ecosystems介接方法
